# 카페 사장님 설치 가이드

컴퓨터를 잘 몰라도 따라 할 수 있게 만들었습니다. 10분이면 됩니다.

## 준비물
- 노트북 1대 (가게에 둘 것)
- 모니터 1대 + HDMI 케이블 (손님이 보는 번호판용)
- 가게 와이파이 공유기

---

## 1단계: 파이썬 설치 (최초 1회)
1. https://www.python.org 접속 → "Download" 클릭
2. 설치 시 **"Add Python to PATH"** 체크박스를 꼭 켜세요 (중요!)
3. 설치 완료

## 2단계: 프로그램 내려받기
1. 이 깃헙 페이지에서 초록색 **"Code"** 버튼 → **"Download ZIP"**
2. 압축 풀기 (바탕화면 등)

## 3단계: 실행
- **윈도우**: `start.bat` 더블클릭
- **맥**: 터미널에서 `bash start.sh`

처음 실행하면 라이브러리를 자동 설치하느라 1~2분 걸립니다. 이후엔 빠릅니다.

## 4단계: 화면에 뜨는 정보 확인
실행하면 검은 창에 이런 게 나옵니다:
```
손님 주문 주소 : http://192.168.0.5:8000/
QR 코드 파일   : qr_order.png
주방 관리창    : http://192.168.0.5:8000/kitchen?token=xxxxx
관리자 토큰    : xxxxx
```
- **QR 코드 파일**(`qr_order.png`)을 인쇄해서 테이블마다 붙이세요
- **주방 관리창** 주소를 사장님 노트북 브라우저에 즐겨찾기 하세요
- **관리자 토큰**은 남에게 알려주지 마세요 (사장님 전용 열쇠)

## 5단계: 모니터 연결
1. 노트북에 HDMI로 모니터 연결
2. 모니터 쪽 브라우저에서 **손님 화면**(`/display`) 열고 **F11**(전체화면)
   → 손님이 주문번호 "준비중 / 완료"를 봅니다

---

## 매일 사용법
- **손님**: 테이블 QR 찍고 주문 → 자동으로 번호 발급
- **사장님**: 주방 관리창에서 [제조완료] 클릭 → 손님 화면에 "완료" 표시
- 손님이 받아가면 [픽업완료] 클릭 → 번호판에서 사라짐

## ⚠️ 보안 주의 (꼭 읽어주세요)
- **공유기 "포트포워딩"을 켜지 마세요.** 가게 안에서만 쓰면 외부 해킹 위험이 없습니다.
- 가게 밖에서도 매출을 보고 싶다면, 포트포워딩 대신 **Tailscale** 같은 안전한 방법을 쓰세요 (별도 안내).
- 관리자 토큰은 사장님만 아세요. 손님은 주문/번호판만 볼 수 있고 완료 버튼은 못 누릅니다.

## 자주 묻는 질문
**Q. 노트북을 껐다 켜면 주문이 사라지나요?**
A. 아니요. 그날 주문은 저장돼 있어 다시 켜면 복구됩니다.

**Q. IP 주소(192.168...)가 매번 바뀌나요?**
A. 공유기 설정에 따라 가끔 바뀔 수 있습니다. 바뀌면 QR을 다시 인쇄하세요.
   (고정 IP 설정법은 docs/고정IP설정.md 참고 — 선택사항)

**Q. 여러 대 노트북에서 같이 쓸 수 있나요?**
A. 1대를 서버로 두고, 다른 기기는 브라우저로 그 주소에 접속하면 됩니다.
